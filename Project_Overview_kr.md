# 프로젝트 기획서: 글로벌 K-컬처 디지털 웹 매거진 (Web Magazine)

## 1. 프로젝트 주제 및 선정 배경
* **주제:** 전 세계 사용자를 대상으로 K-문화, 예술, 트렌드, 로컬 라이프스타일을 소개하고 교류하는 다국어 지원 프리미엄 디지털 웹 매거진 플랫폼
* **선정 배경:**
  최근 K-팝, K-드라마를 넘어 한국의 전통문화와 독창적인 로컬 크리에이터들에 대한 전 세계적인 관심이 급증하고 있습니다. 하지만, 이러한 문화적 자산들을 단순히 나열하거나 언어 장벽 때문에 깊이 있게 전달하지 못하는 한계가 존재합니다. 이에, 오프라인 백화점(Department Store)의 '공간의 층(Floor)'이라는 감각적인 큐레이션 방식을 웹에 차용하여, 누구나 직관적이고 몰입감 있게 K-컬처를 경험할 수 있는 디지털 쇼룸이자 문화 교류의 장이 필요하다고 판단하여 기획하게 되었습니다.

## 2. 기획 의도
커머스나 예약과 같은 딱딱한 목적 기반의 플랫폼에서 벗어나, 방문자가 오직 **'문화적 영감(Inspiration)과 미적 경험(Aesthetic)'**에 집중할 수 있도록 고안했습니다. 단순한 안내 페이지를 벗어나 한 편의 인터랙티브 화보집을 넘겨보는 듯한 사용자 경험(UX)을 구축하며, 실 실시간 자동 번역 시스템을 통해 국적과 언어를 불문하고 글로벌 팬과 아티스트가 온전히 교감할 수 있는 '글로벌 문화 허브 매거진'을 지향합니다.

## 3. 기존 유사 서비스와의 차별성 (핵심 경쟁력)
기존의 K-컬처 홍보 웹사이트나 정보 포털(예: 방문의 해 공식 홈페이지, 일반 웹진 블로그 등)과 비교하여 본 프로젝트가 갖는 차별성은 다음과 같습니다.
* **1. 정보 나열형이 아닌 '공간 탐험형(Spatial Exploration)' 큐레이션:**
  일반적인 웹진이 텍스트 위주의 '목록(List)' 형태를 띠는 반면, 본 플랫폼은 백화점의 1층부터 6층까지 각 테마 공간을 돌아다니는 듯한 시각적 메타포를 제공합니다. 사용자는 목적 없이 방문해도 화보와 영상을 따라 유기적으로 문화를 탐험하게 됩니다.
* **2. 텍스트를 압도하는 '오디오-비주얼(Audio-Visual) 인터랙션':**
  딱딱한 썸네일 구조를 탈피하고, 진입부터 전체 화면 숏폼/비디오 배경과 사운드 제어 기능을 결합했습니다. 스크롤마다 화면이 딱 맞아떨어지는 자석 스크롤링과 Framer Motion 기반의 시네마틱 트랜지션은 하이엔드 럭셔리 브랜드의 룩북에서나 볼 수 있는 프리미엄 감각을 제공합니다.
* **3. 플러그인을 넘어선 '네이티브 수준의 무손실 다국어 렌더링':**
  단순 구글 번역기 위젯을 띄운 타 사이트와 달리, UI/UX 디자인 자체가 다국어(영어, 일본어, 중국어 등) 텍스트 길이 변화에 깨지지 않도록 (Responsive Scaling) 초기부터 설계되었습니다. 번역 이질감 없이 글로벌 사용자가 자국 언어로 작성된 프리미엄 잡지를 읽는 듯한 경험을 보장합니다.

## 4. 교육훈련 내용과의 연관성
본 프로젝트는 **프론트엔드 웹 개발(React 기반 SPA 구축) 및 실무 프로젝트 수행 능력**을 배양하는 교육훈련 과정의 핵심 목표와 정확히 부합합니다.
* **1. 최신 프론트엔드 생태계 실무 활용 (React & Vite):**
  빠른 빌드 시스템인 Vite와 React 18 버전을 사용하여, 실제 현업에서 요구하는 최신 SPA(Single Page Application) 개발 환경을 세팅하고 구조 설계 능력(컴포넌트 분리, 훅 사용)을 증명합니다.
* **2. 고도화된 UI/UX 구현 능력 (Tailwind CSS & Framer Motion):**
  교육을 통해 학습한 CSS 레이아웃 제어 능력을 바탕으로, 실무에서 각광받는 Tailwind CSS를 도입해 초고속 반응형 웹을 구현했습니다. 특히 Framer Motion 라이브러리를 통해 학습 수준을 넘어선 완성도 높은 애니메이션 및 인터랙티브 웹(자석 스크롤 등) 개발 역량을 확보했음을 보여줍니다.
* **3. 클라이언트 사이드 데이터 연동 및 전역 상태 통제:**
  `react-router-dom`을 통한 동적 파라미터 라우팅(`/:category`, `/detail/:id`), SearchParams 처리 능력을 구현했으며, `i18next`를 통한 애플리케이션 전역 상태 관리 및 다국어 환경 제어 능력을 포트폴리오에 반영했습니다.
* **4. 독립적인 산출물 완결 능력 (Mock Data 기반 컴포넌트화):**
  백엔드가 종속되지 않은 상태에서도 프론트엔드 로직만으로 `mockData.ts`를 구조화하여, 실무적인 컴포넌트 단위 테스트 및 렌더링, 페이지 조립을 완수하는 주도적인 프론트엔드 아키텍처 역량을 함양했습니다.

## 5. 프로젝트 내용
공간(1F~6F)이라는 메타포를 활용하여 콘텐츠를 큐레이션하는 에디토리얼 특화 플랫폼입니다.
* **몰입형 메인 큐레이션 (Hero & Live Media):** 풀스크린 비디오와 자석 스크롤링(Snap Scrolling)을 통해 첫 화면부터 압도적인 시각적 몰입감을 제공합니다.
* **문화 달력 (Culture Timeline):** 이달의 주목할 만한 K-컬처 및 오프라인 행사를 다이내믹한 팝업 UI와 함께 커버스토리 형식으로 하이라이트합니다.
* **층별 전시 가이드 (Floor Guide):** 트렌드, 팝업 교류, 공연/전시, 로컬 등 각 층별 고유 테마를 가진 가상 문화 갤러리를 제공합니다.
* **에디토리얼 & 화보 뷰어:** 단순 리스트가 아닌 지그재그(Z-pattern) 레이아웃 기반과 대형 여백을 살린 아름다운 온라인 기사 형태로 콘텐츠를 노출합니다.
* **실시간 글로벌 번역:** 언어 설정 옵션을 통해 화면의 모든 활자와 소개글을 즉시 번역하여 제공합니다.

## 6. 활용 장비 및 재료 (기술 스택)
* **Frontend:** React 18, TypeScript, Vite
* **Styling & Animation:** Tailwind CSS (반응형 유틸리티), Framer Motion (고급 트랜지션 및 인터랙션)
* **Routing & State:** React Router DOM v6
* **Internationalization (국제화):** `i18next` 패키지(실시간 다국어 지원)
* **Design Concept:** Glassmorphism(블러 기반 투명도 UI), High-Contrast Editorial Layout

## 7. 프로젝트 구조
빠르고 시각적으로 수려한 렌더링에 집중한 독립형 정적 웹 애플리케이션 구조입니다.
* **페이지(Pages):**
  - `LandingPage`: 히어로 비디오, 캘린더, 아티스트/브랜드 스포트라이트가 결합된 전면 스크롤 매거진 뷰
  - `FloorIntroPage`: 선택한 층(테마)의 분위기와 에디터의 기획 의도를 설명하는 랜딩 페이지
  - `FloorContentPage`: 고품질 화보와 텍스트 기사 조각을 잡지처럼 나열하는 에디토리얼 아티클 컨테이너
  - `DetailPage`: 개별 문화, 브랜드 인터뷰 전문을 읽고 이미지를 딥다이브하는 상세 아티클 뷰어
* **컴포넌트 설계:** 컴포넌트를 기능별로(레이아웃, 홈섹션, 공통 번역 모듈 등) 완전히 쪼개어 배치/조립과 유지보수가 용이하도록 구성

## 8. 활용 방안 및 기대 효과
* **글로벌 진출용 디지털 포트폴리오:** 국내의 잠재력 있는 디자이너, 장인, 아티스트, 로컬 브랜드들이 자신들의 스토리와 예술성을 가장 트렌디하고 글로벌 친화적인 방식으로 소개할 수 있는 창구가 됩니다.
* **언어 장벽 제로의 한국 문화 사이트:** 관광객이나 해외 K-팬들에게 마치 웅장한 디지털 박물관이자 미디어 아트를 보는 듯한 경험을 선사하여 한국 문화 진입 장벽을 대폭 낮춥니다.
* **강력한 브랜딩 플랫폼(B2B 마케팅 활용):** 단순한 회사 홈페이지나 쇼핑몰 템플릿의 형태에서 벗어나, 특화된 하이엔드 브랜드 프로모션, 페스티벌 캠페인 사이트, 박람회 공식 웹진 등의 서식으로 범용적인 상용화 파급력이 매우 큽니다.
